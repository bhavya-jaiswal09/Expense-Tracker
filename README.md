# Expense Tracker

A simple and modern Expense Tracker web application built using React.js and Vite.

## Features

- Add Income and Expenses
- View Total Balance
- Edit/Delete Transactions
- Search and Filter Transactions
- Expense Analytics with Charts
- Responsive UI
- Data stored using LocalStorage

---

## Tech Stack

- React.js
- Vite
- React Router DOM
- Context API
- Tailwind CSS
- Recharts
